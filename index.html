Site da RAM Soluções

O projeto se baseou no desenvolvimento de um site expositivo para que fosse possível expor os celulares que a empresa reforma, mostrando as especificações de cada aparelho e o valor para o mesmo, possibilitando que o cliente entre em contato diretamente com a empresa via WhatsApp para que ele adquira o mesmo.

É também possível que o cliente acompanhe a manutenção de seu aparelho pelo site, através de um código que é dado para o mesmo.

É possível acessar o site em www.ramsolucoes.com.br.

Sistema do Grupo Podium

O projeto que ainda está em andamento se baseia na criação de um sistema de cursos para o Grupo Podium, empresa da área de cursos de variados setores. Sendo assim, a partir do sistema, os alunos devem conseguir realizar seus cursos tendo acesso às aulas com textos, imagens e audios, às apostilas, às práticas e aos testes de cada aula, podendo ao fim, retirar seu certificado.

Além disso, o sistema abrange também toda a área administrativa da empresa, podendo gerenciar alunos, cursos, pacotes, contratos, presenças, mensalidades, licenças e etc. Abaixo estão relatadas em imagens algumas dessas funcionalidades.
